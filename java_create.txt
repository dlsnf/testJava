1. Java 개발 환경 설정
먼저 Java 개발 환경을 설정해야 합니다.

JDK 설치

Oracle JDK 또는 OpenJDK를 다운로드하고 설치합니다.
설치 후, JAVA_HOME 환경 변수를 설정합니다.
VS Code 설치 및 플러그인 설치

VS Code를 설치합니다.
Java Extension Pack (Java 개발을 위한 확장팩)을 설치합니다.






2. Java 샘플 프로젝트 생성
Java 프로젝트 생성

VS Code에서 Ctrl+Shift+P를 눌러 명령 팔레트를 엽니다.
"Java: Create Java Project"를 선택합니다.
프로젝트 템플릿 선택 (예: "No build tools" 또는 "Maven").
프로젝트 폴더 위치와 이름을 지정합니다.
코드 작성

기본적으로 src 폴더에 App.java 또는 Main.java가 생성됩니다.
